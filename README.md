# Учебник по языку Go

Добро пожаловать в учебник по языку программирования Go! Этот репозиторий содержит полный курс, охватывающий все темы — от базового синтаксиса до продвинутых концепций, необходимых эксперту.

## Описание

Учебник рассчитан на широкую аудиторию:
* **Новички** — найдут пошаговое введение в язык, установку окружения, основы синтаксиса и типов данных.
* **Продвинутые разработчики** — освоят параллельное программирование (concurrency), микросервисы, работу с базами данных, сетевое программирование и обобщенное программирование (generics).
* **Эксперты** — углубятся в устройство языка под капотом: сборщик мусора, escape analysis, профилирование, рефлексию, кодогенерацию и интеграцию с C/C++ через CGO.

## Содержание

Учебник состоит из 17 глав:

1. **Введение в язык Go** — что такое Go, первая программа, инструментарий.
2. **Основы языка Go** — переменные, типы данных, операции, массивы, условия, циклы.
3. **Функции** — параметры, возврат значений, анонимные функции, замыкания, рекурсия.
4. **Указатели** — работа с памятью, указатели и функции, производительность.
5. **Производные типы и структура памяти** — структуры, методы, срезы, карты, выравнивание памяти.
6. **Пакеты и модули** — импорт, модули, зависимости, vendor, создание модулей.
7. **Строки** — руны, операции со строками, оптимизация через `strings.Builder`.
8. **Интерфейсы и Архитектура** — интерфейсы, type assertion, duck typing, композиция.
9. **Параллельное программирование и Concurrency** — горутины, каналы, мьютексы, паттерны конкурентности.
10. **Обработка ошибок и Управление ресурсами** — `error`, `defer`, `panic`/`recover`, `context`.
11. **Потоки и файлы** — `Reader`/`Writer`, работа с файлами, буферизированный ввод-вывод.
12. **Сетевое программирование и Микросервисы** — TCP/UDP, HTTP, REST API, gRPC.
13. **Базы данных** — `database/sql`, транзакции, MySQL, PostgreSQL, SQLite, MongoDB.
14. **Обобщенное программирование (Generics)** — параметры типов, ограничения, универсальные структуры.
15. **Промышленное тестирование** — unit-тесты, бенчмарки, fuzzing, моки.
16. **Производительность под капотом** — GC, escape analysis, `pprof`, метрики, OpenTelemetry.
17. **Системный уровень и Метапрограммирование** — рефлексия, `go generate`, `unsafe`, CGO.

Подробное оглавление доступно в файле [`00_toc.md`](00_toc.md).
